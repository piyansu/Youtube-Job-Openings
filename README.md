   ```bash
   cd tripmate
   npm install
   ```

3. **Install frontend dependencies**
   ```bash
   cd client
   npm install
   ```

4. **Add your `.env` file** in the root of the backend folder with:
   - MongoDB URI
   - Cloudinary credentials
   - Mapbox token, etc.

5. **Run the backend**
   ```bash
   npm start
   ```

6. **Run the frontend**
   ```bash
   cd client
   npm start
   ```
